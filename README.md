<h1 align="center">Sistema de Folha de Pagamento em Java OO</h1>

<p align="center">
  <strong>Folha de Pagamento Deitel</strong> é uma aplicação desenvolvida em <strong>Java puro</strong>, inspirada nos exercícios do livro 
  <strong>Java: Como Programar</strong>, de Deitel & Deitel.
</p>

---

## ✨ Sobre o Projeto
Este projeto foi desenvolvido como estudo prático dos princípios de Programação Orientada a Objetos (POO) apresentados no renomado livro Java: Como Programar (Deitel & Deitel).

## Objetivo
Implementar um sistema robusto de gestão de folha de pagamento, com:
- ✔ Cálculo automático de salários conforme regras específicas por tipo de funcionário
- ✔ Controle flexível de diferentes categorias empregatícias
- ✔ Modelagem orientada a objetos com boas práticas de encapsulamento, herança e polimorfismo

## Funcionalidades Principais
- Cálculo dinâmico de pagamentos (horistas, assalariados, comissionados)
- Extensibilidade para novos tipos de funcionários
- Demonstração de conceitos como:
    - Classes abstratas
    - Interfaces
    - Sobrescrita de métodos
    - Relacionamentos entre objetos

---

## 🚀 Funcionalidades
- **Cadastro de Funcionários**: Suporte para criar funcionários assalariados, horistas e comissionados.
- **Cálculo de Salários**: Implementa cálculos personalizados (ex.: horas extras para horistas, comissões para comissionados).
- **Demonstração de POO**: Uso de herança, polimorfismo e encapsulamento em um caso prático.

---

## 🛠️ Tecnologias Utilizadas

- **Java SE**: Linguagem principal, sem dependências de frameworks.
- **Paradigma**: Programação Orientada a Objetos (POO).
- **Ferramentas Recomendadas**:
    - IDE: [IntelliJ IDEA](https://www.jetbrains.com/idea/) ou [Eclipse](https://www.eclipse.org/).
    - Build: JDK 11 ou superior.

## ▶️ Como Executar
1. Clone o repositório
```bash
git clone https://github.com/carlos0ff/folha-de-pagamento-java.git
```
2. Configure o ambiente
   - Instale o [JDK](https://www.oracle.com/java/technologies/downloads/?er=221886) (versão 11 ou superior).
   - Abra o projeto em sua IDE preferida (IntelliJ IDEA, Eclipse ou VS Code com extensões Java).

3. Compile e execute:
   - Compile: ```javac src/main/java/*.java```
   - Execute: ```java src/main/java/Main.java```
4. **Saída**: A aplicação exibirá a lista de funcionários e seus salários no console.
